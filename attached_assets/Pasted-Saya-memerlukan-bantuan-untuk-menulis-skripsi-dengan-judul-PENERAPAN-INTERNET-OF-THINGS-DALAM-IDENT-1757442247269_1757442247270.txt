Berikut detail lengkap penelitian dan project yang sudah saya kerjakan:

KONTEKS PENELITIAN:
Project ini adalah sistem IoT real-time untuk monitoring kondisi jantung menggunakan ESP32 yang terhubung dengan sensor biomedis (MAX30105 untuk heart rate/SpO2, MLX90614 untuk temperature) yang mengirim data ke Firebase Realtime Database, kemudian diproses menggunakan algoritma Naive Bayes untuk klasifikasi kondisi jantung (Normal, Kurang Normal, Berbahaya) dan ditampilkan melalui web application React dengan dashboard real-time. Sistem ini sudah fully implemented dan berjalan dengan akurasi 100% pada testing data.

TEKNOLOGI YANG DIIMPLEMENTASIKAN:

Hardware: ESP32 microcontroller dengan sensor MAX30105 (heart rate, SpO2), MLX90614 (temperature), LCD display 16x2
Firmware: Arduino IDE dengan kalibrasi tekanan darah (-15 untuk sistolik, -10 untuk diastolik)
Database: Firebase Realtime Database untuk real-time data streaming dan PostgreSQL untuk training data storage
Machine Learning: Algoritma Naive Bayes dual implementation (client-side TypeScript dan server-side Node.js) dengan 26 training samples
Frontend: React + TypeScript dengan Vite, menggunakan Radix UI components, TanStack Query untuk state management
Backend: Express.js + TypeScript dengan API endpoints untuk classification, training data management, evaluation
Real-time Features: Firebase subscription hooks untuk live data updates, notification system untuk kondisi berbahaya
ALGORITMA NAIVE BAYES IMPLEMENTATION:
Sistem menggunakan Gaussian Naive Bayes dengan 6 parameter input (suhu, BPM, SpO2, tekanan sistolik, tekanan diastolik, signal quality) untuk mengklasifikasikan kondisi jantung. Implementasi dual: client-side untuk real-time processing dan server-side untuk API endpoints. Training data terdiri dari 26 samples dengan 3 kelas output. Model mencapai 100% accuracy pada evaluation dengan precision dan recall per kelas. Feature importance analysis menunjukkan kontribusi setiap parameter dalam classification decision.

FITUR SISTEM YANG SUDAH IMPLEMENTED:

Dashboard real-time dengan live vital signs display dan Naive Bayes classification results
Training module untuk manage training data dan retrain model
Evaluation module dengan confusion matrix, precision/recall analysis, ROC curve
Data collection protocol dengan informed consent form dan subject management
Historical data visualization dengan charts (temperature, heart rate, SpO2, blood pressure)
